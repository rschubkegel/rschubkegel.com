Assignment

The purpose of this assignment was to create an original website using only HTML and CSS. The project requirements are as follows:


  • The site needs to have at least three pages
  • Each page needs to reference a single CSS file
  • Images must be used
  • There needs to be at least one table
  • There needs to be at least one list
  • Each page needs to pass W3 HTML validation
  • There must be a clear, intentional theme across all the pages

Design

I chose to create something simple and clean-looking, as this was my first time making my own webpage. Typical lorem ipsum placeholder text is used on the other pages. While it lacks interesting content, it was an effective project to practice my newly-acquired skills.
